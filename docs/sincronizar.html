Puente de sincronización · AyudaVE

Genera el “favorito” (bookmarklet) que un voluntario en Venezuela usa para copiar la lista de la fuente a AyudaVE — desde su propio navegador, con su consentimiento.

Requisito previo: el endpoint de AyudaVE tiene que ser público por HTTPS con certificado válido. Hoy el DNS aún no apunta a 136.68.45.14, así que ayudahumanitariavenezuela.com no resuelve todavía. Apunta el DNS (o, temporalmente, abre el ingress de Cloud Run y usa la URL *.run.app) antes de repartir el bookmarklet.

1 · Configura

No lo cambies salvo que uses la URL *.run.app mientras el DNS propaga.
Es el mismo token que protege /api/admin/ingest. Trátalo como una contraseña: solo voluntarios de confianza.

2 · Arrastra a tu barra de marcadores

Arrastra este botón a la barra de favoritos del navegador (o clic derecho → añadir a marcadores):

★ Sincronizar AyudaVE

3 · Úsalo (el voluntario, en Venezuela)

  1. Abrir desaparecidosterremotovenezuela.com y esperar a que cargue la lista.
  2. Pulsar el favorito ★ Sincronizar AyudaVE.
  3. Aparece un panel abajo a la derecha con el progreso. Al terminar: “✅ Sincronización completa”.
